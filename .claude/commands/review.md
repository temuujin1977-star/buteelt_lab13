# /review

Энэ project-ийн одоогийн өөрчлөлтийг security, robustness, maintainability талаас review хий.

Шалгах зүйлс:

- URL validation зөв эсэх
- Expired link зөв шалгаж байгаа эсэх
- SQL injection эрсдэл байгаа эсэх
- Error response internal detail ил гаргаж байгаа эсэх
- Edge case test дутуу байгаа эсэх
- Code structure route/service/repository layer-ийг дагаж байгаа эсэх

Гаралтыг severity дарааллаар, файл/мөрийн reference-тэй бич.
